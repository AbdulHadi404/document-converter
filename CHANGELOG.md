# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-31

### Added
- Initial release of Document Converter
- Premium quality conversion (95-99% accuracy)
- Full content extraction (text, images, tables)
- Automatic image extraction from DOCX and PDF
- OCR support for scanned PDFs (300 DPI)
- Pandoc integration for best DOCX quality
- Enhanced PDF processing with pdfplumber
- Fallback support (Mammoth for DOCX)
- Comprehensive documentation
- Windows batch script for easy execution
- Support for 99+ documents with 469+ images extracted

### Features
- DOCX to Markdown conversion (95-98% quality with Pandoc)
- PDF to Markdown conversion (80-88% quality, 85-90% with OCR)
- Image extraction and proper referencing
- Table conversion to markdown format
- Formatting preservation (bold, italic, headings, lists)
- Relative image paths for portability
- Organized output structure
- Detailed logging and progress tracking
